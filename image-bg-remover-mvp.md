## 1️⃣ 产品概述

### 1.1 产品定位
一款基于 Cloudflare Workers 和 Remove.bg API 的轻量级图片背景移除工具，无需存储、即传即用。

### 1.2 核心价值
- ✅ **零存储**: 图片仅在内存中处理，不落地存储
- ✅ **快速部署**: 基于 Cloudflare 边缘网络，全球访问
- ✅ **低成本**: 利用免费额度，初期几乎零成本
- ✅ **简单易用**: 拖拽上传，一键下载

### 1.3 目标用户
- 需要快速移除图片背景的普通用户
- 电商从业者（产品图处理）
- 设计师（快速抠图）
- 自媒体运营（素材处理）

## 2️⃣ 功能需求

### 2.1 核心功能（P0 - 必须有）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **图片上传** | 支持拖拽或点击上传，格式：JPG/PNG/WebP | P0 |
| **背景移除** | 调用 Remove.bg API 处理图片 | P0 |
| **结果预览** | 实时显示处理后的透明背景图片 | P0 |
| **图片下载** | 一键下载 PNG 格式结果 | P0 |
| **错误提示** | API 失败、格式错误等友好提示 | P0 |

### 2.2 辅助功能（P1 - 应该有）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **进度反馈** | 上传/处理中显示加载状态 | P1 |
| **图片限制** | 前端校验大小（≤25MB）和格式 | P1 |
| **响应式设计** | 适配移动端和桌面端 | P1 |
| **CORS 支持** | 允许跨域调用（如需嵌入其他页面） | P1 |

### 2.3 未来功能（P2 - 可以有）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **批量处理** | 一次上传多张图片 | P2 |
| **历史记录** | 本地存储最近处理记录（不上传服务器） | P2 |
| **自定义尺寸** | 选择输出图片分辨率 | P2 |
| **API 开放** | 提供公开 API 供第三方调用 | P2 |

## 3️⃣ 技术架构

### 3.1 架构图

```
┌─────────────┐      ┌──────────────────┐      ┌──────────────┐
│   用户浏览器  │ ───→ │ Cloudflare Worker │ ───→ │ Remove.bg API │
│  (前端页面)  │ ←─── │  (边缘计算节点)   │ ←─── │  (背景移除)   │
└─────────────┘      └──────────────────┘      └──────────────┘
                              │
                              ↓
                         （内存处理，不存储）
```

### 3.2 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | HTML + CSS + Vanilla JS | 单文件嵌入 Worker |
| **后端** | Cloudflare Workers | JavaScript 运行时 |
| **API** | Remove.bg | 背景移除服务 |
| **部署** | Wrangler CLI | Cloudflare 官方工具 |
| **存储** | 无 | 纯内存处理 |

### 3.3 数据流

```
1. 用户上传图片 → 2. Worker 接收 FormData → 3. 转发至 Remove.bg API
                                                        ↓
5. 用户下载 ← 4. Worker 返回 Blob (image/png)
```

## 4️⃣ 用户流程

### 4.1 主流程

```
访问网站 → 上传图片 → 自动处理 → 预览结果 → 下载图片
   │          │           │           │           │
   ↓          ↓           ↓           ↓           ↓
 加载页面   格式校验     调用 API    展示 PNG    浏览器下载
```

### 4.2 异常流程

| 场景 | 处理方式 |
|------|----------|
| 图片格式不支持 | 前端提示"仅支持 JPG/PNG/WebP" |
| 图片超过 25MB | 前端提示"图片大小不能超过 25MB" |
| Remove.bg API 失败 | 显示错误信息（配额用尽/网络问题） |
| 网络超时 | 提示"处理超时，请重试" |

## 7️⃣ 风险与限制

### 7.1 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| Remove.bg API 不稳定 | 服务不可用 | 添加重试机制、错误提示 |
| Cloudflare 冷启动 | 首次请求慢 | 使用预热请求 |
| 大图片处理超时 | 请求失败 | 前端限制大小、后端设置超时 |

### 7.2 业务风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| API 配额用尽 | 服务中断 | 监控用量、及时升级 |
| 滥用攻击 | 成本激增 | 添加 IP 限流、验证码 |
| 版权争议 | 法律风险 | 添加使用条款、免责声明 |

### 7.3 限制说明

- 单张图片最大 25MB（Remove.bg 限制）
- 免费账户每月仅 50 张免费处理
- 不支持批量处理（MVP 阶段）
